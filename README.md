# -chapter 5计算机组成
###组成：cpu中央处理单元  主存储器  输入／输出系统
cpu(算数逻辑单元／寄存器／控制单元)
主存储器（按层次结构分：寄存器／高速缓冲存储器／主存）
IO系统（非存储设备／存储设备）

###子系统连接：cup和存储器通过总线（数据总线／地址总线／控制总线）连接；
               cup和IO设备通过接口（控制器）连接，例如HDMI  USB  火线   SCSI
###程序执行：周期包括取指令、译码、执行
